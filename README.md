# SpaceXRockets
The app contains information about SpaceX rockets and a list of their launches.

## Technology stack
* SwiftUI
* Combine
* URLSession(+Combine)
* MVVM

## Requirements
* iOS 16.0+
* Xcode 15.2

## Tasks
1. Add settings module
2. Add shimmer
3. Network error handling
4. Cache using SwiftData
5. Refactoring of filtering of unsuccessful flights
6. Add Preview
